This page demonstrates the new "View Cookies" feature in the DPDPA consent widget that shows detailed cookie information similar to Privy.

Features Implemented:

  • View Cookies button in the consent widget
  • Displays cookies grouped by category (Necessary, Functional, Analytics, Marketing, Social, Preferences)
  • Shows detailed information for each cookie:
    • Cookie name
    • Host/domain
    • Third-party indicator
    • Expiration period
    • Purpose description
    • Provider information
  • Modal popup with scrollable content
  • Loading state while fetching cookie data
  • Graceful handling when no cookies have been scanned
  • Cached API responses for performance

How to Test:

  1. Click the button below to trigger the consent widget
  2. In the consent widget, click the "View Cookies" button
  3. The modal will display detailed cookie information (if available)
  4. If no cookies have been scanned yet, you'll see a friendly message

Technical Details:

The feature uses the following components:

  • New API endpoint: /api/cookies/domain-cookies
  • Modified DPDPA widget with View Cookies button
  • Modal popup with categorized cookie display
  • Integration with existing cookie scan data
