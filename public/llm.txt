# Consently - DPDPA 2023 Consent Management Platform

## Overview
Consently is India's premier DPDPA 2023 compliant consent management platform. We help Indian businesses achieve comprehensive data protection compliance through automated cookie scanning, multilingual consent management, real-time compliance reporting, and privacy preference management.

**Website**: https://www.consently.in  
**Platform Type**: SaaS (Software as a Service)  
**Target Market**: Indian businesses of all sizes (SMBs, Enterprises, E-commerce, SaaS companies, Healthcare, Finance, Education)  
**Compliance Standard**: Digital Personal Data Protection Act (DPDPA) 2023

## API & Integration

### RESTful API Endpoints

#### Cookie Management
- `GET /api/cookies/scan` - Cookie scanning endpoint
- `GET /api/cookies/scan-enhanced` - Enhanced scanning with storage detection
- `GET /api/cookies/widget-config` - Widget configuration management
- `GET /api/cookies/widget-public/[widgetId]` - Public widget endpoint
- `GET /api/cookies/widget-stats/[widgetId]` - Widget statistics
- `GET /api/cookies/analytics` - Cookie analytics
- `GET /api/cookies/compliance` - Compliance scoring
- `GET /api/cookies/policy-generator` - Generate cookie policy
- `GET /api/cookies/categories` - Cookie categories management
- `GET /api/cookies/consent-analytics` - Consent analytics
- `GET /api/cookies/consent-log` - Consent logging

#### DPDPA Management
- `POST /api/dpdpa/consent-record` - Record consent
- `GET /api/dpdpa/activities` - Data processing activities management
- `GET /api/dpdpa/widget-config` - Widget configuration
- `GET /api/dpdpa/widget-public/[widgetId]` - Public widget
- `GET /api/dpdpa/widget-stats/[widgetId]` - Widget statistics
- `GET /api/dpdpa/analytics` - DPDPA analytics
- `GET /api/dpdpa/compliance-report` - Generate compliance report
- `GET /api/dpdpa/grievances` - Grievance management
- `GET /api/dpdpa/purposes` - Purpose management

#### Privacy Centre
- `GET /api/privacy-centre/preferences` - Get privacy preferences
- `PATCH /api/privacy-centre/preferences` - Update preferences
- `GET /api/privacy-centre/preferences/history` - Consent history
- `GET /api/privacy-centre/pages` - Pages where user interacted
- `POST /api/privacy-centre/rights-requests` - Create rights request
- `GET /api/privacy-centre/rights-requests` - Get rights requests

#### Blog & Content
- `GET /api/blog` - List blog posts (with pagination, filtering, search)
- `GET /api/blog/[slug]` - Get individual blog post
- `GET /api/blog/categories` - Get blog categories
- `GET /blog/rss.xml` - RSS feed

#### Analytics & Reporting
- `GET /api/analytics/dashboard` - Dashboard analytics
- `GET /api/reports/analytics` - Generate reports

#### User Management
- `GET /api/user/profile` - Get user profile
- `PUT /api/user/profile` - Update profile
- `PATCH /api/user/profile` - Update email/password

### Webhook Events (Enterprise)
- `consent.granted` - When user grants consent
- `consent.withdrawn` - When user withdraws consent
- `rights.request.created` - When data subject rights request is created
- `rights.request.completed` - When rights request is completed
- `grievance.created` - When grievance is filed
- `grievance.resolved` - When grievance is resolved
