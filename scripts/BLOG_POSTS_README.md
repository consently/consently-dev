## How to Insert Blog Posts

### Option 1: Using Supabase SQL Editor

1. Open your Supabase project dashboard
2. Navigate to SQL Editor
3. Copy and paste the contents of `insert-blog-posts.sql`
4. Run the SQL script
5. Verify posts are created in the `blog_posts` table

### Option 2: Using psql Command Line

```bash
psql -h your-db-host -U your-username -d your-database -f scripts/insert-blog-posts.sql
```

### Option 3: Using Supabase CLI

```bash
supabase db execute -f scripts/insert-blog-posts.sql
```
